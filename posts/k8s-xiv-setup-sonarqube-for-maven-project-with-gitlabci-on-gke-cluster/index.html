Setup Sonarqube for Maven Project With Gitlab-CI on GKE Cluster

Posted on 2020, Aug 08 by hoangmnsd One min read
Table of Contents
  • Giới thiệu
  • Yêu cầu
  • Cách làm

Giới thiệu

Tưởng tượng rằng bạn đã tạo GKE Cluster, dựng Gitlab Runner, Gitlab self-hosted.
Bạn đã có 1 java project build bằng Maven trên Gitlab. Bạn đang sử dụng Gitlab CI để tạo pipeline.
Bài này hướng dẫn cách setup 1 server Sonarqube để scan quality code cho 1 project sử dụng Maven (ngôn ngữ là Java Spring Boots).
Kết thúc bài này sẽ có 1 flow kiểu như sau, project Java của bạn mỗi khi có thay đổi sẽ
trigger Gitlab Runner thực hiện scan quality code bằng Sonarqube,
sau đó build artifacts (tạo file .jar, .war),
build Docker images
và publish Docker images đó lên Registry của bạn (có thể là Dockerhub hoặc GCR, etc…)
